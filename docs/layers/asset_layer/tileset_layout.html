Tileset Layout

Tilesets are an atlas of tiles placed down on the map. Split into 8 columns of 32x32 pixel tiles.

There’s technically a limit to how many rows you can have, but you’re unlikely to hit it. For reference, Fading Memory has a tileset with 332 rows, which seems fine so far.
