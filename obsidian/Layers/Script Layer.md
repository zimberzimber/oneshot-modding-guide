Unlike hell, the devil resides one layer above the deepest, and his name is Ruby.
This layer opens up a lot of possibilities, to the point where you can make something that's completely different from vanilla OneShot.

If you're here just for the minor script modifications required for [[Save Directory]] or some aspects of the [[Editor Layer]], then you'd only care about [[Modifying Scripts]].

- Basic Ruby understanding. Look up an introductory guide if you're new.
